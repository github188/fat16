关于damingdan.com

大名单是一款可以在线进行各类排名分享的轻型社区类网站, 站点会收集各类流行与有意思的排名. 他可以用于各类决策的参考以及 各类娱乐性的话题素材源.

各类名称的字数限制

榜名: 50个汉字 藤名: 50个汉字

关于大名单的代码风格

JS代码风格 类名: 驼峰式 class StudentSubmitForm

一般变量名: 混合式 var studentSubmitForm

函数名称: 混合式 function getDamingdanData(){ // To do something }
